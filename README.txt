Steps to run this project:

s1. open in command prompt from the "TASKMANAGER" folder
s2. To run backend  (Flask)
   -> cd backend
   -> python app.py       #http://127.0.0.1:5000/tasks

s3. To run Frontend  (React js)
   -> cd Frontend
    -> cd fapp
   -> npm start           #http://localhost:3000/
